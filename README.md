# Portfolio Website (GitHub Pages)

This is a simple, fast portfolio site built with **HTML/CSS/JavaScript**.

## Customize it

Open `index.html` and replace:

- Your name (“Your Name”)
- Email (`you@example.com`)
- GitHub + LinkedIn links
- The “About” text

Open `script.js` and replace the `projects` list with your real projects.

Optional:

- Put your resume in this folder as `resume.pdf` (or update the link in `index.html`).

## Preview locally

Easiest option:

- In Cursor/VS Code, right-click `index.html` and use a live preview extension, or just open the file in your browser.

## Publish on GitHub Pages

1. Create a new GitHub repository (example: `portfolio`).
2. Add these files to the repo (at the repo root):
   - `index.html`
   - `styles.css`
   - `script.js`
   - `README.md`
   - `resume.pdf` (optional)
3. Push to GitHub.
4. In GitHub: **Settings → Pages**
   - **Build and deployment → Source**: “Deploy from a branch”
   - **Branch**: `main` and `/ (root)`
5. Wait 1–2 minutes, then GitHub will show your site URL in the Pages section.
